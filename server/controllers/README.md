# IPFS file UPLOAD in NodeJs

## Testing API
1. Clone repo
```bash
git clone https://github.com/mondyfy/ipfsupload.git
```
2. change directory
```bash
cd ipfsupload
```

3. Install dependencies
```bash
npm i
```
or 
```bash
yarn
```

4. Copy .env.example and update API keys
```bash
cp .env.example .env
```

5. Start Server
```bash
npm run start
```
